Point Of Sale Order On Website

You can easily track your Point Of Sale Order on website

Point Of Sale Order On Website Extension for ODOO POS store helps users to easily track their Point Of Sale Order history on the website portal.

This Extension Allows

  • Separate menu for POS Orders.
  • List of all POS Orders in pagination.
  • View a single POS Order record on a website.
  • Pdf invoicing of each POS Order.
  • Quick and Easy Installation.

  • Website View
